Public IP Reused to Pass through

註冊Q-Balancer與後方網段要共同使用的IP，當某個IP在Link setup中的Link IP Binding及pass through ip registration 都有註冊時，則該IP還是可以在Q-Balancer後方的設備使用。Q-balancer實體上不會 binding 該IP，但是若在Q-Balancer有對該IP作服務對應的設定，則當該服務封包經過Q-Balancer時，Q-Balancer將會使用這個IP所設定的功能。這個功能可以讓Q-Balancer與後方設備共用IP。
選取ISP，填入IP，再按【Register】。
