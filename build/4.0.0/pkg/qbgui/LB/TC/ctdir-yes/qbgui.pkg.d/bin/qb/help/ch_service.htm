Service Config.

Q-Balancer對於Service的控管與Mapping除了一般的Port Redirect之外，還支援了定義Protocol的能力
這裡能夠針對單一的Service或是一群Services建立群組，這些群組就是定義給Q-Balancer的Service/Firewall功能的參數

Service : Service Group的名稱
Type : "Known" 表示為系統預設的Service，"User" 表示為自行定義
Protocol : 定義該ServiceGroup使用的Protocol，除了預設之TCP、UDP、ICMP，還有自行定義Protocol No.的欄位(Defined)，例如常用的VPN Protocol為Protocol 50、Protocol 51
Port : 設定TCP or UDP Protocol使用的Port No
