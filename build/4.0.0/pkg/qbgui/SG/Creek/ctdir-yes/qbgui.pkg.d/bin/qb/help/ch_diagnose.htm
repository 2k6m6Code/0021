Diagnosis Tools

Q-Balancer針對線路除錯提供了相當有用的各種功能

Ping 發出ICMP封包
Ping by Arp Packet 用Arp Broadcast的方式發出ICMP封包，通常用於偵測與Q-Balancer介接的Device
Packet Captor 抓取封包的工具，使用的參數如同Tcpdump
Show Arp Cache Show Q-Balancer上的Arp Table
Query QOS Status 顯示QOS各項運作資訊
Restart QB Engine 重新啟動QB的程式，不是Reboot

packet captor的常用參數

-i Interface, Port 1 use eth0, Port 2 use eth1...
-n Not display Protocol Name, show port no.
Host IP Designate the HOST, ex: host 168.95.1.1
Port No. Designate the Port, ex: port 80
And Intersection
Not Exclusion
