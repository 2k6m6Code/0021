Service Config.

Other than using existing services, Q-Balancer allows you to define protocols of your own.
Here, we can group services with multiple port or a single port. These service groups become the parameters for selection in Service/Firewall.

Service : The name of the service group.
Type : "Known" represents a built-in service. "User" means user-defined.
Protocol : Define the protocol of the service group. Other than tcp, udp, icmp, you can also fill in custom Protocol No. in the "defined" field. For example, the common VPN protocol are Protocol 50 and Protocol 51.
Port : Set port number for TCP or UDP protocol
