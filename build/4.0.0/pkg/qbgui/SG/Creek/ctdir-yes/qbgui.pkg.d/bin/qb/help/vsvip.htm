Link IP Binding

This is the interface to bind extra IP addresses to the box. You can also add IP addresses in batch by click the [ ? ] button.
Select the ISP, then click in the blank filed, the correspondence IP address will appear, and you can change it directly.



How to add multiple IPs in Batch Way

Usage: IPSET_1 ; IPSET_2 ... ; IPSET_N

where IPSET_n := { FORMAT1 | FORMAT2 | FORMAT3 }

FORMAT1 := ip

FORMAT2 := start ip : number of ip

FORMAT3 := start ip - end ip
