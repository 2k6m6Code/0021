Layer7 Log Configuration

Layer7 Log Configuration can log layer7 packets.You can select the logs for Layer 7 applications to analyze.

Method:

  1. Go to Analysis Tools->Syslog Support and check the box Enable QB Traffic Log

  2. Check the box Enable QB Log on Local

    3. You can also check Enable Remote Syslog Server to export the layer 4/7 traffic logs.

  3. Click Apply
