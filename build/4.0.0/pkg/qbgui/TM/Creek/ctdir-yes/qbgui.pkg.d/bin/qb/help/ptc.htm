Setup Policy-Based Filter

Let a policy in Policy Routing be regulated by a class in Class Definition.

  1. Browse Filters

    2. Click on a policy to see which class it is regulated by.

  2. Configure a Filter

    3. If the bandwidth of a policy should be regulated, please click on the Edit icon of the policy. There will be a list of applicable classes defined in Class Definition shown in the screen below. Click on the desired class and press OK.

  3. Remove a Filter

    4. To remove a filter, please edit the intended policy and set all ISPs to default class.

When you edit a policy and selecting a class, you only need to modify the entries of the ISP paths this policy takes.
