Administration Console

Reset ARP cache ... Sent requests asking equipments hooked to Q-Balance to reset MAC addresses associated with Q-Balancer's current IP address.
Set Gateway Permanent ... Set MAC addresses of gateways permanently in Q-Balancer's cache.
Maximum User : The maximum number of users logged in at any given time.
Edit User Account : Edit Q-Balancer user account. An account can only log on once at any given time.
Domain Name Resolver : Q-Balancer's own DNS setting. To send e-mails out, please specify.
Remote Config. Management : Import or export configs to another site. Please refer to section [Configuration Management]
History Export FTP Server : Export policy history to an FTP server.
Set Date : Set system date and time.
Time Zone : Set time zone.
Alarm E-mail : Notification e-mail address for any change in link status.

[Configuration Management]

  1. Save / Load Config in Local

    2. The save / load config. file option is located on the top of the user interface.
    Please click [Load], a new pop-up will windows appears. At least three default configuration sets will be shown: active, boot, and default.
    Active Current read-only configuration set in edit. It is rarely used.
    Boot The configuration used after reboot.
    Default The factory default configuration. This is read-only.
    These three configuration set cannot be deleted.

  2. Export / Input Config From Remote

    3. Please select [Remote Config]. You will see Remote Config. Tools.
    Backup Config. Download config. files via http
    Upload Config. Upload config. files via http
    Upload Upgrade Package Upload firmware via http
    Launch Upgrade Process Launch the firmware update
