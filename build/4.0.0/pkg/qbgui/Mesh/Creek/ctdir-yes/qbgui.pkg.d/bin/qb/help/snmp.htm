Snmp Support

  • Enable SNMP (recommended):
    Start SNMP service. You can use SNMP management software to see Q-Balancer network status in SNMP v1/v2 MIB.

    When SNMP trap is checked, Q-Balancer would send out SNMP v1 trap to SNMP manager IP address. Currently, Q-Balancer can send ColdStart, WarmStart, LinkDown, LinkUp messages.
